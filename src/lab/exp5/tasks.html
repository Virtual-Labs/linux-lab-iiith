Create a test folder in your home directory and change into this directory. Add five text files to the directory. You may create some text files or copy the files created in other experiments to this folder.

Task 1:
Archive the files inside the folder using Zip commands.

Task 2:
Archive the same set of files using Tar command.

Task 3:
Compress the files using Bzip2. While using Bzip2, make sure you preserve the original file as well.

Task 4:
Compress the files using Gzip2.

Task 5:
Create another directory direct. Use Unzip command to decompress the files with .zip extensions inside the new directory.

Task 6:
Extract the Tar files into the new directory.

Task 7:
Decompress the .gz files into the new directory using Gunzip command.

Task 8:
Decompress the .bzip2 files into the new directory using Bunzip2 command.
