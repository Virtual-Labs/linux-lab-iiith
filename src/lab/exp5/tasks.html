Create a test folder in your home directory and change into this directory. Add five text files to the directory. You may create some text files or copy the files created in other experiments to this folder.

Task 1:
Archive the files inside the folder using zip and tar commands.

Task 2:
Compress the files using Bzip2, Gzip and Zip utilities. While using Bzip2, make sure you preserve the original file as well.

Task 3:
Create another directory direct. Unzip all the above compressed and archived files into the directory direct using the commands - unzip, Bunzip2 and Gunzip - for the respective file extensions.
