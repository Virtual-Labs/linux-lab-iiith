8. Searching
Introduction

Objective

The objective of this experiment is to familiarize the users with the commands to search files in the machine using command line functions. After finishing the Experiment, the user would be able to use the search functions available in Linux easily.

Theory

Grep Command

Grep is an extremely powerful utility. It can not only search for files and folders in the entire file system but also searches for strings within the files.

Regular Expressions

Regular Expressions provide a flexible way to match the strings or patterns of characters. Options

Grep can be used with the following options:
v - verbose mode which displays the
A
B
C
o

Using with other Commands

Grep can also be used in conjunction with other commands using the pipes to get the desired outputs.
grep (Regular expression, -v, -A, -B, -C, -o), find (-iname, -delete)
