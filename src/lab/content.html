System and Software Virtual Lab
Welcome to Systems and Software Virtual Lab
Introduction

This lab covers all the knowledge and skills required for systems administration on a Linux system, including the management of files, processes, users, text processing, version control, networking, backup, encryption and decryption of data, by using a Linux based command line interface. The experiments follow a practice-based approach and are designed in such a way so as to provide a theoretical understanding of the various tasks, followed by a hands-on practical experience of performing these tasks.

List of experiments
  1. File Management
  2. Process Management
  3. Basic Networking Concepts
  4. Version Control with Subversion
  5. User Management
  6. Examining Log and Configuration Files
  7. Advanced File Management
  8. BASH and Environment Variables
  9. Rsync
  10. GPG Encryption and Decryption
  11. Getting Help
  12. Status Commands
  13. Text Processing
  14. Executing Commands

Pre-requisites

Familiarity with the basic computer systems. Programming is not a pre-requisite.

Objective

The goal of this lab is to increase the students' knowledge and practice of basic, but essential tasks of system administration. This skill will be valuable in both increasing the student's productivity in working with (Linux based) computers, and in stimulating his/her enthusiasm and interest in systems related software engineering.

How to Use Lab

The lab is divided into theory and experiment sections.

  • First go through the theory section for an overview of the commands and concepts that will be required to perform the experiment.
  • Then proceed to the experiment section.
  • Register and obtain your virtual-labs user ID and password.
  • Login using your virtual-labs login credentials.
  • In the terminal, set the IP address to 10.2.48.12.
  • Login using your virtual labs user ID and password.
  • Now, perform the tasks using the suitable commands.
