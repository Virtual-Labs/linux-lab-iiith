System and Software Virtual Lab
Introduction

Welcome to Systems and Software Lab! This lab provides introduction to Linux shell for performing basic system operations such as file management, text editing, permissions, searching and basic networking. Information on working with compressed files ang getting system information is also provided.

The objective of this lab is to make students familiar and comfortable with working in a Linux environment. This lab also promotes basic understanding of system administration processes by providing a hands-on experience with a browser based access to Linux command-line interface.

How to Use Lab

There are 9 experiments in the lab. Each experiment is divided into theory and experiment sections. The theory section provides a brief overview of the various commands available. The users are expected to try using these commands on their machine for better understanding. The experiment section provides a list of tasks for the user to perform in the shell provided along side.

To use the shell, please follow these steps:

  • Register and obtain your virtual-labs user ID and password.
  • Login using your virtual-labs login credentials.
  • In the terminal, set the following credentials: 
    Host / IP or SSH URL : 10.2.48.12
Port : 22
  • Login using your virtual labs user ID and password.
  • You can use the shell to execute the tasks listed alongside.

List of experiments
  1. File Management
  2. Text Editor
  3. File Content
  4. File Permissions
  5. Archival and Compression
  6. Getting System Information
  7. Pipes and Redirection
  8. Searching
  9. Basic Networking
Target Audience

Graduate and Undergraduate students of CS or any other streams.

  • CS05566 UNIX PROGRAMMING in JNTU
  • CS1002 UNIX INTERNALS in Anna University
  • EURCS511 UNIX PROGRAMMING LAB in GITAM University.
  • CS0IU4312 UNIX and Shell Programming Lab in Vignan University.

Pre-requisites

In order to use this lab, the audience is assumed to have familiarity with the basic computer systems and should be enthusiastic to work on a Linux environment.
