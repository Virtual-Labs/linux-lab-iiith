1. Create a new file called "lazydog" and type the following text into it: "The quick brown fox jumped over the lazy dog." You need to go to insert mode to type content into the file.
  2. Save the file and observe the message near the bottom of the screen. If you don't see any message, try saving again. If you accidentally closed the file, reopen it.
  3. Copy the line and paste it 5 times. Now paste the same line 1000 times. Turn the line numbering on.
  4. Search for the word "brown". Turn the search highlighting on. Press n to move to the next occurence of the search term. Press N to move to the previous occurence of the search term.
  5. Delete 1000 lines.
  6. Replace "brown" with "black" in all the lines where it occurs.
  7. Replace all black's with brown's using undo command.
  8. Replace all brown's with black's using redo command.
