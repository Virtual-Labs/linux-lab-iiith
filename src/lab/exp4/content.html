Experiment 4: File Permissions
Introduction

Welcome to the permissions experiment! This experiment is intended to provide understanding of Linux file permissions. This experiment also involves understanding of various command(s) which can be used to modify permissions.

The experimental platform for this experiment is a command line interface which mimics the terminal of a linux machine. The command line interface can help in learning the basics of file permissions. A series of tasks which can be performed on available command-line interface is also provided.

Please proceed to the theory section to get an overview of the commands. Then move to the experiment section to get some hands-on experience. After you are done with the experiment, go to the quiz section to test your understanding.

Objective

The goal of this experiment is to explain various types of Linux file permissions and their significance. The experiment also has theory and tasks related to commands which can be used to change file permissions.
