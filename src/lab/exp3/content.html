Systems and Software Virtual Lab
Experiment 3: File Content
Introduction

Welcom to File Content Experiment, the third experiment of systems and software lab. This experiment consists of description of various unix command programs which helps in viewing content of text files in different dimesnsion according to requirement.

This platform of this experiment is command line interface which give terminal of linux machine. Linux terminal will help user to perform basic linux commands for the tasks mentioned in simulation section.

Theory section will give brief descriptions of commands with examples which helps practicing to view content of a text file in different ways using linux commands.

Objective

The goal of this lab is to teach you some basic ways to view content of a text file in different formats or dimensions using unix commands.

Theory

Below are various linux commands which help in getting content of a text file in different dimesnsions.

1.echo

echo is a built in command in the bash and C shells that writes its character strings to standard output. More description of its usage and option will be available in man page( type in terminal $man echo)

Syntax of echo command

 username@hostname:~$ echo

We shall see how to display sample by using echo command.

To display current day date, enter the command echo $(date) 

swaroop@Kundeti:~$ echo "Hello World"
Hello World

2.cat

cat is a unix command which is used to display content of a text file.

cat can be used in multi ways like "display text files on screen", "copy text files", "combine text files", create new text files". More decription of usage and its options can be found in man pages.

Syntax of cat command

username@hostname:~$ cat [filename]

We shall see how to use cat command for displaying content of text file.

To know the content of file example.txt 

swaroop@Kundeti:~$ cat example.txt 
this a example text file.

3.wc

wc will display a count of lines, words, and characters in a file.

Syntax of wc command

username@hostname:~ wc [options] [file]

Below are the available options for wc command

OptionPurpose
cCounts and displays number of bytes in a file
mCounts and displays number of characters in a file.
lCounts and displays the number os lines in a file
wCounts and display number of words in a file

We see how to display count of lines, words and characters in a file using wc command 

swaroop@Kundeti:~$ wc example.txt 
1  5 26 example.txt
 
Above output is :
1  = Lines
5  = Words
26 = characters

4.head

Head is unix command which is used for print first few lines of a text file to standard output by default.

Syntax of head command

 username@hostname:~head  [file-name]

Below are the available options for head command

OptionsPurpose
c the N number of bytes from the initial part of file.
n The N number of lines from the initial part of file.

We shall now see how to display a file content using head command 

	  swaroop@Kundeti:~$ head example.txt 
	  this a example text file.

Note: By default head command will display the first 10 lines of a file. The autpu can be customized by using options available using head command. More information can be found in man page.

5.tail

Tail is a unix command which is used to display last few lines of a text file. More information about its usage with options can be found in man pages.

Syntax of tail command

username@hostname:~tail [file-name]

We shall see how to display last few lines of a text file content using Tail Command 

swaroop@kundeti:~$ tail example.txt
this a example text file.
The same content was copied to example2.txt
	  
swaroop@Kundeti:~$ tail -n1 example.txt 
The same content was copied to example2.txt

6.Diff

diff is a unix command which comapares line by line and outputs the differences between two files. More information about its usage and its options can be found in man page.

Syntax of diff command

username@hostname:~$ diff  [file-name1] [file-name2]

We shall see how to compare two files using diff command 

swaroop@Kundeti:~$ diff example.txt example2.txt 
2d1
The same content wass copied to example2.txt

7.sort

Sort is a standard unix command that prints the lines of its inputs or concatinated files in its argument list in a sorted order. sorting is done on based on one or more sort keys extracted from each line of input. More information about usage of sort and its options can be found in man page.

Syntax for sort command

username@hostname:~$ sort [file]

We shall see how to sort a file which containes phone book

swaroop@Kundeti:~$ cat  phonebook
Mahesh, Kumar   9491787823
Swaroop, Kumar  9866223311
Kiran, Kumar    9988998899

Above is the sample phone book file, we shall sort it by alphabetical order using sort

swaroop@Kundeti:~$ sort phonebook Kiran, Kumar 9988998899 Mahesh, Kumar 9491787823 Swaroop, Kumar 9866223311

8.more

More is a command line program which is used to view content of text file as one screen at a time, hence it is considered as terminal pager. More will allow only forward navigation through a large file. Further information about its usage and options can be found in man page.

Syntax for more command

username@hostname:~$ more [file-name]

Below is a an example to view apache.conf whichis huge text file. using more command 

swaroop@kundeti:~$ more /etc/apache2/apache2.conf

9.less

less is similar to more command which is used to view content of text files. less command will navigate in both ways forward as well as reverse. Further information about its usage and options can be found in man page.

Syntax for less command 

username@hostname:~$less [file-name]
