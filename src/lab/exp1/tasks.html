Tasks

  1. Delete all the contents of your home directory.
  2. Create a directory by name "demo" and an empty file by name "demo.txt"
  3. List the contents of the home directory along with their properties.
  4. Create 5 directories - a,b,c,d,e in the current directory. Then, move them such that the following hierarchy is created:
    • Directory a/ is in your home directory.
    • Directory b/ is in a/.
    • Directories c/ and d/ are in directory b/.
    • Directory e/ is in directory d/.
  5. Create 5 files named 1,2,3,4,5 in the current folder. Now copy them into previously created a,b,c,d,e respectively.
  6. Delete the entire directory structure (including the files) you've created.
