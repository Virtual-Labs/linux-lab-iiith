Tasks

  1. Find out how much RAM is used (in Mega Bytes) by the Linux machine which you are running now.
  2. Using ps command, check out the list of processes which are running as currently logged in user.
  3. Using top, check which process/command is taking more CPU resource and as which user it is running.
  4. Check out who are the currently logged in users, at what time they logged in and what is the application they are running currently.
  5. Check out which Linux distribution you are in currently and the complete kernel release information.
  6. Check out who is the vendor, what is the model name of the processor you are currently running on and also check how many core/sibling processors it has inside it.
  7. Check out how much memory space is used in the root file system drive(/) and the drive under home directory (/home).
  8. Display the current date and calendar of previous, current and next month of the current year. Also checkout how long the system is running.
