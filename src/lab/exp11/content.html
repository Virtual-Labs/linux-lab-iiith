Getting Help
Introduction

The experiment aims at introducing the commands which have been provided for the ease of user. These commands provide a detailed manual on any command used in Linux. Linux provides a detailed documentation of the various commands and their attributes and how they can be used. The commands introduced int his experiment help invoke that description.

Objective

The objective of this experiment is to introduce the commands which invoke the documentation of the commands used in the Linux system.

Theory

man

'man' is the command to invoke the documentation on the commands. When you type the 'man' command followed by an argument, it searches the documentation manual for the command entered in the argument. Type 'man man' to open up the documentation on 'man' command. 


        johndoe@laptop:~$ man emacs

It opens a manual page for emacs. The output is something like this:

info

'info' is similar to 'man'. It was implemented as a replacement to man pages in the early '90s. Info pages provide a more complete and comprehensive documentation than the man pages. Its usage is straightforward.

Type 'info' without any arguments to display the various options and all the utilities covered by the command. For the info page of a utility, just type 'info' followed by the name of the utility. 


        johndoe@laptop:~$ info emacs

It opens an information page on emacs. The output is something like this:

whatis

'whatis' displays the list of man pages available for a particular command. It the manual page names and displays them as a result of the command. There is also the option of using wildcards (-w) and regular expressions (-r). 


	johndoe@laptop:~$ whatis emacs
	emacs (1)            - GNU project Emacs

The '-h' option

Any command followed by '-h' option displays the help page of the command.

apropos

The 'apropos' command searches the man page names and descriptions against the search keyword and displays all the pages which include the keyword. It is useful for finding the right man pages and associated commands. Like the 'whatis' command, you can enter wildcards (-w) and regular expressions (-r) as keywords.

Bash autocomplete

Bash also provides an autocomplete option. Once you type part of the command on the command line, hit the [TAB] key and Bash autocompletes the rest of the command name. In case of more than one possibilities, Bash returns the set of possibilities.
